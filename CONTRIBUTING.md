# Contributing

Thanks for helping make this list better. Good contributions keep it trustworthy and useful.

## What we accept

We add resources that are:

- **Specific.** They solve a real prep problem, not "10 tips to ace any interview."
- **Current.** Interview formats change fast. Stale advice gets removed.
- **Useful.** Someone prepping this week would actually open it.
- **Honest.** We label anything self-promotional clearly.

## How to add a resource

1. Fork this repo.
2. Add your resource to the right section, in the existing format.
3. Keep the description to one punchy line. Say what it is and why it helps.
4. Check that your link works and is not a referral or affiliate link.
5. Open a pull request with a short note on why it belongs here.

## Formatting rules

- Use this pattern: `- [Name](url) - short description.`
- Sort by usefulness, not alphabetically. Best resource first.
- One resource per line.
- No duplicates. Search before you add.
- No broken links. Test before you submit.

## What we will not accept

- Low-effort listicles and content farms.
- Paywalled resources with no real free value, unless they are clearly worth it.
- Affiliate or referral links.
- Anything spammy, outdated, or off-topic.

## Removing a resource

If a link dies or a resource goes stale, open a pull request to remove it. Keeping the list clean is as valuable as adding to it.

Thanks for contributing.
